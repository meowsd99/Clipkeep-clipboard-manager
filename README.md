ClipKeep - 智能剪贴板历史管理工具

ClipKeep 是一款基于 PyQt6 开发的现代化剪贴板管理软件，提供强大的剪贴板历史记录、搜索和管理功能。支持文本、图片、文件和富文本内容，具有美观的 WinUI 风格界面和主题切换功能。

✨ 主要特性

- 📋 **全面的剪贴板支持** - 文本、图片、文件路径、HTML 富文本
- 🎨 **现代化界面** - WinUI 3.0 设计风格，支持亮色/暗色/系统主题
- 🔍 **智能搜索** - 实时搜索剪贴板历史内容
- 💾 **数据持久化** - SQLite 数据库存储，数据安全可靠
- 🖼 **图片预览** - 支持缩放和缩略图显示
- 🚀 **性能优化** - 异步加载，避免界面卡顿
- 📱 **系统托盘** - 后台运行，快速访问
- ⚙️ **高度可配置** - 丰富的设置选项

🛠 技术栈

- Python 3.6+
- PyQt6 - 现代化 GUI 框架
- SQLite - 轻量级数据库
- QThreadPool - 多线程异步处理

- 🎯 功能演示

- **主题切换**: 支持系统主题跟随
- **图片管理**: 自动压缩和缩略图生成
- **文件操作**: 拖拽文件路径复制
- **快捷操作**: 丰富的键盘快捷键
- **数据安全**: 本地存储，隐私保护
